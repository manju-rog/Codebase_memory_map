# Mini Bank API

A sample Spring Boot banking application for demonstrating RepoLens AI.

## Features

- User authentication with JWT
- Account management
- Fund transfers
- Transaction history

## API Endpoints

### Authentication
- POST /api/auth/login - User login

### Users
- GET /api/users/me - Get current user

### Accounts
- GET /api/accounts/{id} - Get account details

### Transfers
- POST /api/transfers - Create fund transfer
